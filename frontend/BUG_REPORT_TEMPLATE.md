  - P1
  - P2
  - Not for pilot fix
- Additional notes:

## Severity quick rule
- P1: login, text, file flow, health or data problem that blocks pilot work
- P2: limited issue with workaround, non-destructive, pilot can continue
- Not for pilot fix: cosmetic issue, copy tweak, scope expansion or feature request
